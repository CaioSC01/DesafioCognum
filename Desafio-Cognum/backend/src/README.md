<h2>Instalação do Projeto em suas maquinas</h2>
<h6>Clone este repositorio e apos isso:</h6>

- yarn add ou npm i (serve).
<h6>Apos a instalação:</h6>

- yarn dev ou npm run dev (para rodar o projeto)
